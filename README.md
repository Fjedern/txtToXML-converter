# txtToXML-converter
 
## Fileformat<br/>
P|firstname|lastname<br/>
T|phonenumber|landline<br/>
A|street|city|zipcode<br/>
F|name|birthdate<br/>

P can be followed by T, A and F<br/>
F can be followed T och A<br/>
